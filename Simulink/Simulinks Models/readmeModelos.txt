Se incluyen en esta carpeta los principales modelos de bloques de Simulink usados para la realización de este estudio.

Requisitos para su uso:  se ha trabajado con la versión MatLab R2019 Update 9 (9.7.0.1737446) para Windows de 64-bit (win64).

Concretamente:
- Requisitos operativos: es compatible en Windows (Windows 10,7, Server 2016 y Server 2019), Mac
(10.12,10.13,10.14) y Linux (Ubuntu 18.04, 16.04 LTS, Delbian 9).
Es compatible además con los navegadores Microsoft Edge, Firefox,
Google Chrome y Safari.
- Requisitos de hardware: procesador de 64 bits, memoria mínimo
de 4 GB de RAM (8 recomendados), y espacio en disco mínimo para
la instalación de 3.2 GB.
- Control System Toolbox para el análisis, diseño y sintonización
de los sistemas de control. Sus especificaciones son similares a las de
MatLab.
